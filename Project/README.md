# CampusMarket - Student Marketplace App

## Description
A student marketplace Android app to buy and sell second-hand items, built with Kotlin and Jetpack Compose.

## Features
- Browse listings with category filters
- Post items to sell
- Item detail with seller info
- Search and filter items
- My listings with delete option
- Chat between buyer and seller

## Tech Stack
- Kotlin + Jetpack Compose
- Room Database
- ViewModel + StateFlow
- Navigation Compose
- Sealed classes for UI state
